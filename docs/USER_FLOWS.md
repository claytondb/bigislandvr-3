### Interaction Details

1. **Toggle On/Off**: Click effect name or icon
2. **Adjust Intensity**: Drag slider (0-100%)
3. **Master Volume**: Controls all audio globally
4. **Location Auto**: Effects match location metadata
5. **Custom Mix**: User overrides location defaults

### Effect Types

| Effect | Visual | Audio | Intensity Range |
|--------|--------|-------|-----------------|
| Rain | Canvas overlay particles | Rain sounds | 0-100% (drizzle to downpour) |

| Birds | Occasional silhouettes | Bird calls | 0-100% (distant to nearby) |
| Ocean | None | Wave sounds | 0-100% (distant to shore) |
| Vog | Amber tint overlay | Low rumble | 0-100% (clear to hazy) |
| Mist | Fog particles | None | 0-100% (none to dense) |
